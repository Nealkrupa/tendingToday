🐾 Mascot Widget — Offline Devtools

Fully disconnected from Firebase — no network calls, no real household data touched. firebase.firestore() is replaced below with an in-memory mock (persisted to localStorage under mockFirestoreDB so it survives a real page reload). The actual unmodified mascot.js and achievements.js run against that mock, so what you see here is the real widget code, just fed fake data. Safe to leave open in a separate tab — it never touches tendingtoday-f30e4.

⚠️ You're viewing this via a file:// URL. Firefox blocks <canvas> pixel access (used for skin-color/hat-trim recoloring) on images loaded from local files — this makes the pet's colors look broken here even though the real deployed site (served over https://) is unaffected. To test recoloring locally in Firefox, serve this folder instead of double-clicking the file, e.g. from a terminal in this project folder run python3 -m http.server 8000 and open http://localhost:8000/mascot-devtools.html.
